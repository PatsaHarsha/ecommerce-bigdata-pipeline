# Task 4: Reflection - Big Data Lab 01

4.1 What surprised you most about setting up a Big Data environment?
The extreme sensitivity to version mismatches between Spark Master and Driver was surprising. An minor version difference (Spark 4.0 vs 3.5) caused immediate serialization failures, highlighting that distributed systems require rigorous environment alignment.

4.2 What is the difference between a transformation and an action in Spark? Give one example of each from your notebook.
Transformations are lazy operations that define a data processing plan (e.g., `df.filter()`). Actions trigger the actual computation to produce results or save data. In my notebook, filtering diamonds by cut is a transformation, while `df.show()` and `write.save()` are actions.

4.3 Why does Delta Lake store a delta_log/ folder? What is inside it and why does it matter?
The `_delta_log/` folder contains JSON transaction logs and checkpoints. It provides ACID properties to data in object storage, enabling data integrity, schema enforcement, and "Time Travel" for accessing historical versions.
